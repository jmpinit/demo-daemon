Demo-Daemon: Overview

Demo-Daemon (or DemDae) loads audio-visual JS demos from gists, and reloads code as the gists are updated.

To reload a gist, send a GET request to the server's address with the following format:

http://localhost:3000/update/<gist id>

This can be done by loading the link in a browser or on command line by using curl like so:

curl http://localhost:3000/update/<gist id>

Demo Format

Each demo is an object whose properties are events that get triggered by the demo system.

For example: 
{
	draw: function() {
		var d = 16;
		for(var y=0; y < canvas.height; y+=d) {
			for(var x=0; x < canvas.width; x+=d) {
				var c = (y^x)&255; if(c < 0) c=0; if(c > 255) c=255;
				ctx.fillStyle = 'rgb('+c+','+c+','+c+');';
				ctx.fillRect(x, y, d, d);
			}
		}
	}
}
